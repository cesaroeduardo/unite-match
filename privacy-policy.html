Política de Privacidade

Última atualização:

1. Informações Coletadas

A extensão Unite Match Stats Generator coleta e armazena apenas as seguintes informações:

  • Foto do usuário (opcional): Se você optar por fazer upload de uma foto pessoal, ela será armazenada localmente no seu navegador usando Chrome Storage API. Esta foto é usada exclusivamente para personalizar as imagens de estatísticas geradas.
  • Conteúdo da página: A extensão acessa o conteúdo público da página do UniteAPI.dev para extrair dados de partidas que já estão visíveis na tela. Esses dados são processados localmente e não são armazenados permanentemente.

2. Como Usamos as Informações

Todas as informações coletadas são usadas exclusivamente para:

  • Gerar imagens de estatísticas das partidas do Pokémon Unite
  • Personalizar as imagens com a foto do usuário (quando fornecida)
  • Processar dados localmente no navegador para criar as visualizações

3. Armazenamento de Dados

Todos os dados são armazenados localmente no seu navegador:

  • A foto do usuário (se fornecida) é armazenada usando Chrome Storage API local
  • Nenhum dado é enviado para servidores externos
  • Nenhum dado é compartilhado com terceiros
  • Você pode remover seus dados a qualquer momento limpando os dados da extensão ou desinstalando-a

4. Compartilhamento de Dados

A extensão NÃO:

  • Vende ou transfere dados para terceiros
  • Envia dados para servidores externos
  • Compartilha informações com outras empresas ou serviços
  • Usa dados para publicidade ou marketing
  • Usa dados para análise de crédito ou empréstimos

5. Permissões da Extensão

A extensão solicita as seguintes permissões:

  • activeTab: Para acessar a aba ativa quando você clica no ícone da extensão
  • scripting: Para injetar scripts na página do UniteAPI.dev e processar dados
  • storage: Para armazenar localmente a foto do usuário (opcional)
  • tabs: Para comunicação interna entre componentes da extensão
  • contextMenus: Para criar item no menu de contexto (clique direito)
  • https://uniteapi.dev/*: Para acessar conteúdo do site UniteAPI.dev onde os dados das partidas estão disponíveis

6. Segurança

Implementamos as seguintes medidas de segurança:

  • Todos os dados permanecem no seu dispositivo
  • Nenhuma comunicação com servidores externos
  • Processamento local de todas as informações
  • Armazenamento seguro usando APIs nativas do Chrome

7. Seus Direitos

Você tem o direito de:

  • Acessar os dados armazenados pela extensão
  • Remover seus dados a qualquer momento
  • Desinstalar a extensão, o que remove todos os dados armazenados
  • Optar por não fazer upload de foto pessoal

8. Alterações nesta Política

Podemos atualizar esta política de privacidade ocasionalmente. A data da última atualização será sempre indicada no topo desta página. Recomendamos que você revise esta política periodicamente.

9. Contato

Se você tiver dúvidas sobre esta política de privacidade, entre em contato através do repositório do projeto ou dos canais de suporte disponíveis.

Esta política de privacidade se aplica exclusivamente à extensão Unite Match Stats Generator e não se aplica a sites de terceiros, incluindo o UniteAPI.dev.
